# 问答助手

一个基于 React 的问答助手应用。

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm start
```

应用将在 http://localhost:3000 打开

## 构建生产版本

```bash
npm run build
```

## 项目结构

- `src/components/AIAssistant.js` - 主组件
- `src/components/AIAssistant.css` - 组件样式
- `src/App.js` - 应用入口组件
- `src/index.js` - React 入口文件
- `public/index.html` - HTML 模板
